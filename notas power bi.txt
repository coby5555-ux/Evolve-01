====================================================================================================
                                      ESCENARIO Y MEDIDAS DAX
====================================================================================================

----------------------------------------------------------------------------------------------------
1. CONTEXTO DEL ESCENARIO
----------------------------------------------------------------------------------------------------

Supón que tienes una tabla de ventas con las columnas:
- Fecha
- Región
- Producto
- Cantidad
- Precio Unitario

OBJETIVO: Calcular la venta total por producto.
----------------------------------------------------------------------------------------------------

----------------------------------------------------------------------------------------------------
2. DEFINICIÓN DE MEDIDAS
----------------------------------------------------------------------------------------------------



| TIPO DE MEDIDA             | CÓDIGO DAX                                                           |
|----------------------------|----------------------------------------------------------------------|
| Medida de ventas totales   | Total Sales = SUM(Sales[Quantity] * Sales[Unit Price])               |
|                            |                                                                      |
| Ventas por región y mes    | Sales by Region and Month = CALCULATE([Total Sales],                |
|                            |                               YEAR(Sales[Date]) = 2024,              |
|                            |                               MONTH(Sales[Date]) = 1)                |
----------------------------------------------------------------------------------------------------



====================================================================================================
                                 FUNCIONES LÓGICAS EN DAX
====================================================================================================

----------------------------------------------------------------------------------------------------
1. FUNCIÓN IF (EVALUACIÓN CONDICIONAL)
----------------------------------------------------------------------------------------------------


| EJEMPLO                                                                                          |
|--------------------------------------------------------------------------------------------------|
| Sales Status = IF(Sales[Amount] > 1000, "High", "Low")                                           |
|                                                                                                  |
| EXPLICACIÓN:                                                                                     |
| La función IF evalúa si las ventas son mayores a 1000. Si es cierto, devuelve "High", de lo      |
| contrario, devuelve "Low".                                                                       |
----------------------------------------------------------------------------------------------------

----------------------------------------------------------------------------------------------------
2. FUNCIONES AND / OR (COMBINACIÓN DE CONDICIONES)
----------------------------------------------------------------------------------------------------


| EJEMPLO                                                                                          |
|--------------------------------------------------------------------------------------------------|
| High Sales and Large Order = IF(AND(Sales[Amount] > 1000, Sales[Quantity] > 10), "Yes", "No")    |
|                                                                                                  |
| EXPLICACIÓN:                                                                                     |
| La función AND permite combinar dos condiciones: si las ventas son mayores a 1000 y la cantidad  |
| es mayor a 10, entonces devuelve "Yes", de lo contrario, "No".                                   |
----------------------------------------------------------------------------------------------------

----------------------------------------------------------------------------------------------------
3. FUNCIÓN SWITCH (EVALUACIÓN DE EXPRESIÓN)
----------------------------------------------------------------------------------------------------


| EJEMPLO                                                                                          |
|--------------------------------------------------------------------------------------------------|
| Sales Category = SWITCH(TRUE(), Sales[Amount] > 1000, "High", Sales[Amount] > 500, "Medium",     |
| "Low")                                                                                           |
|                                                                                                  |
| EXPLICACIÓN:                                                                                     |
| Evalúa una expresión y devuelve el resultado que corresponde a un valor de comparación. En este  |
| caso, devuelve las categorías "High", "Medium" o "Low" según el rango de ventas.                 |
----------------------------------------------------------------------------------------------------
